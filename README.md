# LoveAI

LoveAI 是一个以“浪漫表达 + 动态播放 + 作品生成”为核心的 Android 应用。项目从最初的随机浪漫特效播放器，逐步升级为支持方案编排、逐页文案、音乐绑定、收藏再编辑、封面分享和海报导出的作品化产品。

当前仓库已经完成四期建设，已经具备“可编辑、可保存、可复用、可分享”的基础能力。

## 当前阶段

- 当前稳定分支：`codex/phase4-product-upgrade`
- 当前阶段定位：四期完成版
- 四期目标：把项目从“可编辑的浪漫动态生成器”升级为“具备作品沉淀与分享能力的产品”

## 核心模块

- `ui`
  负责启动页、主播放页、方案编辑页、方案库、收藏页、结尾页和页面跳转。
- `viewmodel/LoveViewModel.kt`
  负责当前播放动态、方案状态、收藏状态和自动轮播状态。
- `repository/EffectRepository.kt`
  负责随机特效生成、按类型生成、按变体恢复、按逐页文案生成最终效果。
- `repository/PlanRepository.kt`
  负责方案的保存、查询、复制、最近使用记录、标签、封面和逐页文案持久化。
- `repository/FavoriteRepository.kt`
  负责收藏整套动态，并支持收藏再次转为方案编辑。
- `manager/MusicManager.kt`
  负责扫描内置音乐、导入本地音乐、播放列表管理、切歌与方案绑定。
- `ui/PlanArtworkExporter.kt`
  负责导出方案海报和分享封面。
- `ui/effects`
  负责 12 类浪漫特效绘制、动画节奏和文字表现。

## 当前业务能力

### 1. 启动与主播放流程

- 启动页倒计时进入主页面
- 主页面使用 `ViewPager2` 播放一组动态
- 随机模式下每轮从 12 种特效中随机选 8 个
- 最后一页至少停留 10 秒后进入结尾页

### 2. 特效与文字系统

- 当前支持 12 类浪漫特效
- 支持统一主标题 / 副标题
- 支持逐页单独文案覆盖默认文案
- 已有多种不同的文字布局和视觉风格

### 3. 音乐系统

- 自动扫描 `res/raw` 音乐资源
- 默认顺序循环播放
- 支持上一首 / 下一首 / 播放列表
- 支持方案绑定音乐
- 支持导入本地音乐并写入可复用播放列表

### 4. 方案系统

- 支持方案名称、主标题、副标题编辑
- 支持 12 种特效里手动选择，最少 5 个，最多 8 个
- 支持主题模板
- 支持方案封面风格
- 支持标签
- 支持逐页文案
- 支持方案复制、搜索、筛选、排序、最近使用统计

### 5. 收藏系统

- 收藏的是整套动态，而不是单个页面
- 收藏内容包含：动态序列、文案、标签、绑定音乐、创建时间
- 收藏页支持打开、删除、再次转成方案编辑

### 6. 作品输出与分享

- 支持导出方案海报图片
- 支持分享方案封面
- 海报内容包含：方案封面、标题、副标题、标签、音乐信息、页面列表

## 代码逻辑说明

### 随机生成逻辑

随机模式采用“12 选 8”的策略：

1. 从 12 个特效类型中打乱顺序
2. 取前 8 个类型
3. 每个类型内部随机选一个 `EffectVariant`
4. 生成一轮动态序列

### 方案播放逻辑

当从方案库打开一个方案时：

1. `MainActivity` 读取方案 ID
2. `PlanRepository` 获取完整方案
3. `LoveViewModel` 调用 `loadPlan`
4. `EffectRepository` 按特效类型生成对应页面
5. 若某页配置了逐页文案，则优先使用该页文案
6. 若方案绑定了音乐，则 `MusicManager` 切到指定曲目

### 收藏转编辑逻辑

当从收藏页点击“编辑”时：

1. 读取收藏的 `variantId` 序列
2. 通过 `EffectRepository` 反推出特效类型列表
3. 生成一个新的方案草稿
4. 跳转到方案编辑页继续修改

### 海报导出逻辑

当在方案库点击“导出图”或“分享封面”时：

1. 根据方案封面风格和主题生成渐变背景
2. 把方案标题、副标题、标签、音乐和页面列表绘制到 Bitmap
3. 输出 PNG 文件
4. 导出时保存到应用导出目录
5. 分享时通过 `FileProvider` 暴露只读 URI

## 五期方向

五期不建议再做“小步增强”，而是建议大刀阔斧地把 LoveAI 从“作品型工具”升级成“内容型平台产品”。这期应该从单机工具思维，进入“模板生态 + 智能生成 + 社交传播 + 多媒体输出”的阶段。

### 1. 视频导出正式化

- 不再只导出海报，支持真正的视频导出
- 支持 720p / 1080p 导出
- 支持导出带音乐的 MP4
- 支持横屏 / 竖屏双比例

### 2. 智能生成能力

- 根据主题自动生成一整套方案
- 根据关键词自动生成逐页文案
- 提供不同语气风格：热烈、克制、诗意、纪念、生日、异地
- 支持“帮我续写下一页文案”

### 3. 模板与主题市场

- 内置更多官方主题包
- 节日 / 生日 / 纪念日 / 求婚 / 婚礼等专题模板
- 支持模板预览
- 为后续在线模板更新和素材包分发做结构准备

### 4. 多媒体作品能力

- 支持插入用户照片
- 支持插入短视频片段
- 支持封面图自定义
- 支持音乐 + 图片 + 特效混合编排

### 5. 作品中心升级

- 收藏升级为“作品中心”
- 支持草稿、已发布、收藏、最近编辑分类
- 支持分组、批量管理、回收站
- 支持作品复制、版本历史和回滚

### 6. 分享链路升级

- 一键生成分享卡片
- 生成长图详情页
- 生成二维码或分享口令
- 为未来 H5 打开或网页预览预留能力

### 7. 社交与互动预留

- 作品浏览页
- 评论祝福页预留
- 点赞 / 收藏 / 浏览量预留
- 匿名分享和私密分享两种模式

### 8. 数据与产品化能力

- 用户配置中心
- 作品统计面板
- 首次使用引导与模板推荐
- 错误恢复、导出队列、任务进度反馈

## 推荐五期范围

如果五期真的要“大刀阔斧”，建议直接锁定下面 6 项作为主线：

- MP4 视频导出
- 智能文案与智能方案生成
- 图片 / 视频素材插入
- 模板市场与主题包
- 作品中心与版本历史
- 分享卡片 / 长图 / 二维码

做完这 6 项之后，LoveAI 就不再只是“浪漫特效 App”，而会进入“内容创作平台”的阶段。
